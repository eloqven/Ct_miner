# Code of Conduct

## Purpose

This repository is intended to be a respectful, practical, and focused space for collaboration on NC Wallet Tracker.

## Expected Behavior

Contributors are expected to:

- communicate respectfully and directly
- assume good faith
- give constructive technical feedback
- keep discussions relevant to the project
- respect differences in experience, background, and perspective

## Unacceptable Behavior

The following behavior is not acceptable:

- harassment, insults, or personal attacks
- discriminatory or hateful language
- trolling, intimidation, or repeated disruption
- publishing someone else's private information without permission
- deliberately derailing technical discussion

## Collaboration Standards

When contributing:

- keep pull requests and issues focused
- explain technical decisions clearly
- avoid unnecessary hostility in reviews
- accept that maintainers may reject proposals that do not fit the project direction

## Enforcement

Project maintainers may edit, remove, reject, or moderate contributions and discussions that violate this Code of Conduct.

Repeated or severe violations may result in temporary or permanent exclusion from collaboration spaces related to this project.

## Reporting

If something needs attention, contact the project maintainer through the repository owner account or a private communication channel you already use for this project.

## Scope

This Code of Conduct applies to repository discussions, issues, pull requests, reviews, and other project-related interactions.
