# CT Miner

CT Miner is a React + TypeScript market workstation for:

- tracking a curated crypto universe by category
- entering exact wallet balances per asset
- simulating fee-free trades with a minimum notional
- caching market snapshots locally for portfolio history
- charting cached and fetched price structure
- creating custom alert rules with optional browser notifications

## Stack

- Vite
- React
- TypeScript
- IndexedDB via `idb`
- `lightweight-charts` for candlestick rendering

## Run

```bash
npm install
npm run dev
```
